DevReader

Простой, удобный, а также мобильный справочник по программированию для начинающих

О проекте

DevReader - это небольшой проект, цель которого образовать небольшой ресурс с информацией, которая должна помочь начинающим пронраммистам при освоении какого-либо языка программирования или других вспомогательных компонентов, которые так или иначе связаны с темой программирования.

Идея проекта не заключается в том, чтобы обучить программированию или какому-либо инструменту за короткий срок. DevReader совсем не является точным и практичным учебным пособием по сути. DevReader является просто справочником с необходимой информацией в удобном виде.

Основное преимущество и главная цель проекта - портативность и удобство.

Пользователь, который читает статью в DevReader, видит только текст справочника. Никакой рекламы, никакой ненужной второстепенной информации и никакой "воды" в статьях. Всё просто!

Статья - это каждая отдельная страница по какому-либу языку программирования или инструменту.

Приложения

DevReader доступен с любого устройства, где возможна работа более-менее современного браузера. Скорее всего и вы сейчас читаете это через браузер.

Однако для более удобного чтения у проекта есть приложения ("клиенты"), с помощью которых посещение справочника становится удобнее.

Также через клиент доступны многие другие возможности, недоступные через браузер.

DevReader для Android

Разработка мобильного приложения для Android была начата примерно в то же время, когда был запущен проект DevReader.

Android, в силу дешевизны и относительной простоты разработки приложений, был выбран в качестве основной платформы.

На данный момент приложение находится на стадии предварительного публичного тестирования и в релиз выйдет не раньше релиза проекта.

Уже сейчас можно бесплатно и без подписки загрузить предварительную версию для публичного тестирования и установить приложение на своё устройство.

DevReader для Android на данный момент находится на стадии разработки и в работе приложения могут наблюдаться недочеты и проблемы. Будем рады обратной связи и логам ошибок, если они встречаются.

DevReader для iOS

На данный момент клиента DevReader нет для устройств под управлением операционной системой Apple iOS. Тут очень много факторов, главный из которых - нет разработчика с опытом.

Сейчас ведётся проектирование возможного клиента, разработанного на фреймворке Flutter на языке Dart с кросс-платформенной поддержкой сборки и для Android.

DevReader для Windows 10

DevReader для ОС Microsoft Windows 10 на данный момент находится на стадии прототипирования и пока нет никаких предварительных сборок.

Разработку клиента для Windows 10 планируется начать в ближайщее время, но точных дат пока никто дать не может.

DevReader для Windows 7

В чем разница от версией для Windows 10 и почему бы не использовать одно приложение, чем писать несколько? Резонный и логичный вопрос. Дело в том, что приложения для Windows 10 (в т.ч. и Mobile) разрабатываются на UWP (Универсальная платформа Windows), которая не имеет обратной совместимости с более старыми версиями ОС.

Для Windows 7 клиент будет написан на win32, который поддерживается также в Windows 8, 8.1 и в том же Windows 10, однако в Windows 10 предпочтения отдаются именно UWP-приложениям.

Вероятность того, что разработка приложения для Windows 10 начнется выше, в отличии от приложения для более старых версии ОС. Это всё пока планы. Следите за новостями в нашем Telegram.
